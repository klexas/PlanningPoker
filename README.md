# PlanningPoker
Planning poker implementation in nodeJS and SPA front end. 

Just for fun. Feel free to break or fix !

# Setup


```sh
$ cd node
$ npm install
$ node main.js
```

Port 3001 will be opened to serve the API and 'Client' html site.
